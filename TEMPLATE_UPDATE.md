# Обновление Шаблона nextjs-starter

Чтобы усилить ваш шаблон `nextjs-starter` наработками из этого проекта, выполните следующие действия в репозитории шаблона.

## 1. Новый Скрипт: "Go to Stars" Protocol
Скопируйте `scripts/go-to-stars.ts` в шаблон. Этот скрипт служит для автоматизированного чек-апа проекта перед стартом.

**Установка:**
1. Создайте папку `scripts/` в корне.
2. Положите туда файл `go-to-stars.ts`.
3. В `package.json` шаблона можно добавить скрипт:
   ```json
   "scripts": {
     "go": "npx tsx scripts/go-to-stars.ts"
   }
   ```

## 2. Агентная Инструкция
Скопируйте файл `.agent/workflows/go-to-stars.md`. Это позволит AI-агенту (вам или вашему помощнику) знать, как запускать проверку готовности.

## 3. Базовые Компоненты (Mobile First)
Добавьте компонент `MobileNav.tsx` в базовый набор компонентов шаблона (`src/components/`). Это гарантирует, что любое новое приложение сразу будет иметь нижнюю навигацию для мобильных.

## 4. AI-Ready
Добавьте пакет `@google/generative-ai` в `package.json` шаблона по умолчанию, чтобы "умные" функции были доступны сразу.

```bash
npm install @google/generative-ai
```

## 5. Обновленный CLAUDE.md
Не забудьте обновить `CLAUDE.md` в шаблоне, добавив туда упоминание о запуске `/go-to-stars` в начале каждого нового проекта.
